Active Directory is a system setup for managing a set number of computers. This might be for security or protection. It can also be described as a centralized authentication and permission network resource. Managing permissions and applications can be added with the Active Directory by the administrator. Centralised management is a more secure option, so every network and computer have the same permissions and authorization.

Active Directory was fun to setup, being able manage another user in the domain we created by making passwords and managing its permissions. We first had to make a brand new VM (Virtual Machine) and load on an administator panel which would allow us to add our computers and then this would allow us to manage the permissions. We added an Admin account to our organizational unit and setup the admin account to manage the computers we linked to our domain (Level2.lan).
